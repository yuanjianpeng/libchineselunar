这是一个C语言的库，实现公历和农历的转换。  
它不依赖任何第三方库，甚至不依赖libc。  
支持1901-1-1到2100-12-31的范围。同时支持节气查询。  

编译  
```$ make```

生成一个静态库`libchineselunar.a`，可直接链接到程序中。  

同时生成一个示例程序`calendar`，可以用来显示一个月的日历。
